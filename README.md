# 🌐 Developer Portfolio - Shivang Chaurasia

A modern, responsive, glassmorphic portfolio web application showcasing software engineering projects, technical skills, certifications, training, and achievements.

## ✨ Features

- **Glassmorphism UI**: Built with custom CSS featuring blurred glass cards, gradient text highlights, and smooth hover effects.
- **Fully Responsive**: Mobile-friendly navigation drawer and flexible CSS Grid layouts optimized across devices.
- **Projects Showcase**: Highlights key full-stack, DevOps/cloud, and desktop software projects with direct live demo and repository links.
- **Skills Matrix**: Categorized technical expertise spanning Languages, Frontend, Backend, Databases, Cloud/DevOps, and Developer Tools.
- **Certifications & Achievements**: Displays verified course completion badges, stats, and professional links (GitHub, LinkedIn, HackerRank, Codolio).

## 🚀 Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Styling & Glassmorphic Utilities), Font Awesome Icons
- **Deployment & Hosting**: Works out-of-the-box with GitHub Pages, Vercel, Netlify, or any static Web Server

## 📂 Project Structure

## 🛠️ Quick Start

No complex build steps or dependencies required!

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ShivangChaurasia/Portfolio-MERNTraining.git
   cd Portfolio-MERNTraining
   ```

2. **Open in Browser**:
   Open `index.html` directly in any web browser or use a local server like Live Server in VS Code.
